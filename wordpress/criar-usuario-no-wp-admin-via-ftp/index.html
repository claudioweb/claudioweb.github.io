Criar usuário no wp-admin via FTP

27 JUL
Wordpress
Alguma vez você já teve um cliente precisa de ajuda em seu site WordPress e só possui o acesso ao FTP, mas na verdade não tem nenhuma uma conta do WordPress para administrar? Basta colar este trecho no arquivo functions.php do seu tema atual ou adicioná-lo a uma action, e o WordPress criará automaticamente uma conta de administrador para você usar!

Se já existe uma conta com o nome de usuário ou endereço de e-mail especificado, ele falhará e não ficará registrado.
